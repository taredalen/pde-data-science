1. In terminal 1: 
    ````shell
    rasa run --enable-api --cors "*"
    ````
2. In terminal 2 start server: 
    ````shell
    python -m http.server 8000
    ````
3. Launch once all dependencies are installed:
     ````shell
    yarn serve
    ````
4. Bot enable on : 
    ````shell
     http://172.20.10.3:8081
    ````
